question: if one element is styled by more than one rule, which stylenwill be applicable?
        answer: more specific rule will get more value.
    
GENERAL RULE:
        1. !important will over ride everything.
        2.inline can override everything except !important.
        3. External will work according to specifity.

        SPECIFITY RULE :
        1. # id selector has highest value
        2. * Universal selector has lowest value

        SPECIFITY RULE TRICK (POINT SYSTEM)
        1. 10,000 point to !important
        2. 1000 point to inline
        3. 100 to id
        4 10 to class, attribute or pseudo class
        5 1 for element selector & pseudo element
        6 0 for universal selector
    
        p{
            color:green
        }
         total points=1

        .red-p{
            color:red;
        }
        total points= 10

        div .red-p{
            color:blue;
        }
        total points= 1 + 10 = 11 . so this will apply.
    
        note: - if same rule written two or more times in external sheet then last rule will
        be applicable.
