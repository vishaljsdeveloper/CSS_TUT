colors can be specifies using
        1 predefined color name
        like blue .red etc 
        2 RGB
        rgb(red, green , blue) are base colors
        color value between  0 nd 255
        black: rgb(0,0,0)
        white: rgb(255,255,255)
        red: rgb(255,0,0)
        green: rgb(0,255,0)
        blue: rgb(0,0,255)
        3 RGBA
        rgba(red,green,blue,alpha) alpha is used for giving transparency
        value of alpha=0.0 - 1.0
        0.0=fully transparency
        1.0=non transparent at all
        ex= rgba(255,99,71,0.5)
        4 HEX
        #rrggbb
        hexadecimal values between 00 and ff
        ex= red= #ff0000
        black= #000000
        white= #ffffff
        5 HSL
        hsl(hue, saturation,lightness)
        hue= degree on color wheel 0 to 360
        saturation= percentage value 0% to 100%
        lightness= percentage value 0% to 100%
        ex = red : hsl(0,100%,50%)
        6 HSLA
        same as hsl. bas alpha add ho jata hai isme
